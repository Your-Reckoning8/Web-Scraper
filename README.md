# Web-Scraper
A python tool that pulls real data from websites and analyzes it for security related information. The program connects to a target website and extracts useful information automatically including links, forms, email addresses, and security headers. This tool is used for gathering public information about a website before addressing its security.
*Please note that the current pathway to save the log.txt file is set to my personal pathway. To better fit your needs, you should change this*
